# COS 331 Shell Assignment

## Introduction
This repository contains the source code for the COS 331 course assignment, which involves building a simple functioning shell in C.

## Features
- Prompting the user for a program name
- Executing the specified program in a child process
- Handling multiple arguments to the program
- Proper error handling for input reading, forking, and executing the program

## Files
- `hw2.c`: The main C source code file containing the implementation of the shell.
- `Makefile`: This file contains the makefile. 
- `README.txt`: This file, providing information about the assignment and instructions.
